# 🛒 Oceanx Grocery App

A fast, responsive, and intuitive grocery delivery application built natively for Android. Modeled directly after the functionality and user interface of industry leaders like Blinkit, this app focuses on a seamless quick-commerce experience, taking users from daily essential browsing to checkout in seconds.

## ✨ Features

*   **Lightning-Fast UI:** Smooth, highly responsive interface designed specifically to replicate the rapid navigation of top-tier quick-commerce applications.
*   **Dynamic Product Catalog:** Categorized, easily scrollable inventory for fresh produce, daily staples, and household items.
*   **Smart Cart & Checkout:** Real-time cart updates, intuitive quantity adjustments, and a streamlined, frictionless checkout process.
*   **Search & Filter:** Instantly find products with optimized search functionality.

## 🛠️ Tech Stack

*   **IDE:** Android Studio
*   **Language:** Java / Kotlin
*   **UI Architecture:** Native Android (XML)
